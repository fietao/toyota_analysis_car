==============================================================
CALCULATION RULES
==============================================================
- Share    = row value / Grand Total value (same col)  → store as decimal e.g. 0.524
- MoM      = (curr_month_curr_year - prev_month_curr_year) / prev_month_curr_year
             None/blank if curr month is January (no prev month)
- YoY      = (curr_month_curr_year - curr_month_prev_year) / curr_month_prev_year
             None/blank if prev year value is 0
- YTD      = sum of Jan through curr month
- YTD YoY  = (curr_year_YTD - prev_year_YTD) / prev_year_YTD
- Missing period → write None (blank cell), never 0
