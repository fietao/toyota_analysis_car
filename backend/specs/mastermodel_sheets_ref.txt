  — build_BEV.py reads pipeline_state.json and appends to this table
  — if new_bev_models is empty: prints message and exits cleanly (no error)

  Row 7:     Col E = "ชนิดเชื้อเพลิง", Col F = "Powertrain"  (mapping header)

COLUMNS PYTHON WRITES (cols E and F only):
  Col E: ชนิดเชื้อเพลิง  (Thai fuel type string e.g. "CNG", "ดีเซล", "ไฟฟ้า")
  Col F: Powertrain      (ICE / BEV / HEV / PHEV / N/A)

POWERTRAIN CLASSIFICATION RULES (classify_powertrain in update_raw_data.py):
  "ไม่ระบุ" / "ไม่ทราบ" / "อื่น ๆ"   → N/A
  contains "เสียบปลั๊ก"               → PHEV
  exact "ไฟฟ้า"                       → BEV
  contains "ไฟฟ้า" (not plug-in)      → HEV
  everything else (CNG, diesel, gas)  → ICE

WRITE METHOD: openpyxl (load + save)
  — reads existing col E rows 8+ to find known fuel types
  — finds last used row in col E
  — appends only NEW fuel types not already present
  — saves file after appending
  — if no new fuel types: prints message, closes without saving
