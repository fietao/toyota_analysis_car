# Run The Dashboard

From PowerShell:

```powershell
cd C:\dev\ai-reading-car-analysis\dashboard
npm run dev -- --port 3001
```

Open:

```text
http://localhost:3001
```

## Production Check

```powershell
cd C:\dev\ai-reading-car-analysis\dashboard
npm run build
npm run start -- --port 3001
```

## Useful Checks

```powershell
npm run lint
npm run build
```
